# Patty's Burger Restaurant

A restaurant website built with React and Tailwind CSS.

## Features
- Home page with hero image
- Menu page with add to cart functionality
- Cart with quantity tracking and toast notifications
- About page with image slideshow
- Contact page with form
- Responsive design with hamburger menu on mobile

## Built With
- React
- Tailwind CSS
- Vite
- React Router
